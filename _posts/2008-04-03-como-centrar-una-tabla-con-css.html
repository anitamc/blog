La solución más sencilla es envolver la tabla con un div:

<div class='centraTabla'>

mitabla....

</div>

Y formatear ese div de la siguiente manera:

div.centraTabla{
text-align: center;
}

div.centraTabla table {
margin: 0 auto;
text-align: left;
}

Así de simple :)
